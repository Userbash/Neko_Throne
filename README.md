# Neko Throne

**Neko Throne** is a modern, cross-platform GUI proxy client based on Qt 6. It serves as a powerful frontend for the NekoCore backend, supporting both `sing-box` and `Xray-core`.

[![Build Status](https://github.com/DpaKc404/Neko_Throne/actions/workflows/main_ci.yml/badge.svg)](https://github.com/DpaKc404/Neko_Throne/actions)
[![License](https://img.shields.io/github/license/DpaKc404/Neko_Throne)](LICENSE)

## ✨ Key Features
*   **Unified Backend:** Supports `sing-box` and `Xray-core` out of the box.
*   **Modern UI:** Built with Qt 6.10.2 for a fast and responsive user experience.
*   **Cross-Platform:** Native support for Linux (x86_64, ARM64) and Windows.
*   **Stability First:** Integrated crash handling and safe-mode for containerized/headless environments.
*   **Deep Customization:** Flexible routing rules, theme management, and advanced proxy state tracking.

## 🚀 Getting Started

### Installation
1.  Download the latest release for your platform from the [Releases page](https://github.com/DpaKc404/Neko_Throne/releases).
2.  **Linux:** You can use the portable `.zip` or install the `.deb` package.
3.  **Windows:** Use the installer or the standalone portable version.

### Running in Headless Mode
If you are running in a restricted environment (Distrobox, Docker, or Server), Neko Throne will automatically disable System Tray features to ensure a crash-free experience.
